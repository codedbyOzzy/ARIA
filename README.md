### 🧩 The Convergence Loop
Every interaction with FRIDAY Synapse is no longer a stateless prompt. It is a turn in a continuous **Awareness Loop**.

| Stage | Module | What Happens |
|---|---|---|
| **1. ROUTE** | **ORACLE** | Analyzes complexity to select the optimal model (gpt-4, o4-mini, or local). |
| **2. PREDICT** | **SPECTRE** | Anticipates your next logical step by synthesizing current context. |
| **3. OBSERVE** | **VIGIL** | Monitors real-time user state—energy, focus, and multi-turn goals. |
| **4. RECALL** | **MEMORY** | Pulls factual preferences and project-specific knowledge. |
| **5. CONTEXT** | **THE ARC** | Injects long-term narrative history and definitive decisions. |
| **6. EMOTION** | **ARCHIVE** | Layers in the emotional signature and longitudinal user profile. |

---

### 🧠 The Soul: Deep Identity & Narrative
The core of v1.0 is the **Historical Layer**. FRIDAY doesn't just remember what you said; it remembers what it meant.

- **[THE ARC](https://github.com/codedbyOzzy/the-arc)** — Narrative Tracking. Connects threads across weeks. Detects "Ghost Threads" (reappearing topics).
- **[ARCHIVE](https://github.com/codedbyOzzy/archive)** — Emotional Memory. Builds a profile of your motivators, triggers, and growth over months.
- **[STONES](https://github.com/codedbyOzzy/Intelligence-Stones)** — Adaptive Persona. MindStone & EchoStone tune FRIDAY's voice to your specific frequency.

---

### 🖥️ The Body: Deep Desktop Control
Actual Windows API calls. Not browser tricks. Real power.

```
"Ozan, 3 tura kadar RAM kullanımı %90'ı aşabilir. Chrome'u daraltayım mı?" → SPECTRE Prediction
"Haftalık 'Vision' projesi özetini hazırla" → THE ARC Narrative Retrieval
"Spotify'da lo-fi aç ve sesi %20 yap" → ACTION Stone OS Control
"Bu hatayı debug et (Screen)" → VISION Stone + o4-mini Reasoning
```

- **50+ Integrated Tools** — Spotify, Steam, PowerShell, Playwright, Win32 API.
- **Real Voice Interaction** — Low-latency neural voice pipeline with human-like barge-in.
- **Proactive Engine** — Startup briefings, idle-mode thought surfacing, and system alerts.
